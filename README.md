# Dewise Drink Solutions Corporate Website

This is a modern, responsive corporate website for Dewise Drink Solutions, a consulting and trading firm specializing in comprehensive beverage solutions.

## About Dewise Drink Solutions

Dewise Drink Solutions is a consulting and trading firm specializing in providing comprehensive beverage solutions. Our mission is to guide clients through every stage of beverage creation, from initial concept development to the final bottled product. With a strong foundation in the beverage industry and a commitment to innovation, quality, and sustainability, we deliver tailored solutions that meet the unique needs of our clients.

## Features

- Fully responsive design that works on all devices
- Modern, clean interface with smooth animations
- Fast loading times
- SEO optimized
- Mobile-friendly navigation
- Contact form with validation

## Pages

- **Home**: Overview of services and company highlights
- **About**: Company mission, team, and achievements
- **Services**: Detailed information about all services offered
- **Contact**: Contact form and company information

## Technologies Used

- HTML5
- CSS3 (with custom properties and flexbox/grid)
- JavaScript (ES6+)
- Responsive design with mobile-first approach

## How to Run

1. Clone or download this repository
2. Open `index.html` in your preferred web browser
3. Alternatively, you can serve the files using a local server:

### Python
```bash
# Python 3
python -m http.server 8000
```

### Node.js
```bash
npx serve
```
